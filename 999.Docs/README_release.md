## 개요

ChangeFlow v10.0 시리즈는 2025년 2월부터 12월까지 배포된 주요 버전입니다. 
워크플로우 개선, 보안 강화, 새로운 기능 추가 등 다양한 개선사항이 포함되어 있습니다.
